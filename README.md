## 🧠 Multiplication Brain Game

Ένα διαδραστικό παιχνίδι μνήμης (Memory Match) για την εκμάθηση της προπαίδειας, φτιαγμένο με **Python** και **Streamlit**.

## 🎮 Πώς παίζεται
1. Επιλέξτε τους αριθμούς της προπαίδειας που θέλετε να εξασκήσετε.
2. Πατήστε **🚀 ΞΕΚΙΝΑΜΕ!**.
3. Βρείτε τα ζευγάρια αντιστοιχίζοντας τη λευκή κάρτα (**Πράξη**) με τη γαλάζια κάρτα (**Αποτέλεσμα**).
4. Σκοπός είναι να βρείτε όλα τα ζευγάρια στον λιγότερο δυνατό χρόνο και με τις λιγότερες προσπάθειες!

## ✨ Χαρακτηριστικά
* **360° Card Flip:** Εντυπωσιακά animations κατά την επιλογή των καρτών.
* **Smart Matching:** Διαχωρισμός ερώτησης (λευκή) και απάντησης (γαλάζια).
* **Live Stats:** Χρονόμετρο και μετρητής προσπαθειών σε πραγματικό χρόνο.
* **Responsive Layout:** Λειτουργεί σε υπολογιστές, tablet και κινητά.

## 🛠️ Τεχνολογίες
* Python 3.x
* Streamlit Framework
* Custom CSS3 Animations & Grid Layout
